# Memory Match Loteria

A memory matching game built using JavaScript.

## Technologies Used

- Javascript
- HTML5
- CSS3
- AWS EC2

## Live Demo

Try the application live [here](https://memory_match.arlettepineda.com)

## Features

- User can view game.
- User can flip cards.
- User can match two cards.
